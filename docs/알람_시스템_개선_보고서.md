# 🔔 알람 시스템 개선 보고서

## 📅 작업 일시
2026-01-07

## 🎯 개선 목표
기존의 단순한 비프음 알람을 BBC 스타일 시보음과 시각적 효과로 업그레이드하여 사용자 경험 향상

## ✨ 주요 개선 사항

### 1. **BBC 스타일 시보음** 🎵

#### 기존 방식
- 단순 1200Hz 사인파 0.8초 재생
- 감정적 몰입감 부족

#### 개선된 방식
- **6번의 짧은 비프음** (각 0.1초, 800ms 간격)
- **마지막 긴 비프음** (1초)
- 1000Hz 사인파 사용
- 총 재생 시간: 약 5.8초

```javascript
// BBC 스타일 시보음 구조
비프1 (0.1s) → 대기 (0.7s) →
비프2 (0.1s) → 대기 (0.7s) →
비프3 (0.1s) → 대기 (0.7s) →
비프4 (0.1s) → 대기 (0.7s) →
비프5 (0.1s) → 대기 (0.7s) →
비프6 (0.1s) → 대기 (0.7s) →
긴 비프 (1.0s)
```

### 2. **10초 카운트다운 시각 효과** 🎨

#### 배경색 점진적 변화
- **시작 시점**: 정각/30분 기준 10초 전 (50초부터)
- **변화 과정**: 투명 → 빨간색 (0% → 30% 불투명도)
- **수식**: `intensity = (10 - countdown) / 10 * 0.3`

| 시간 | 카운트다운 | 배경 불투명도 |
|------|-----------|--------------|
| 50초 | 10 | 0% (투명) |
| 52초 | 8 | 6% |
| 55초 | 5 | 15% |
| 58초 | 2 | 24% |
| 59초 | 1 | 27% |
| 00초 | 0 | 30% (최대) |

#### 카운트다운 숫자 표시
- 화면 중앙에 큰 숫자 표시 (120px)
- 흰색 텍스트 + 검은색 그림자
- 펄스 애니메이션 효과 (1초 주기)

### 3. **알람 발동 시 플래시 효과** ⚡

정각/30분 정확히 0초에:
1. **화면 전체 화이트 플래시** (0.5초)
2. **빨간색으로 전환** (30% 불투명도)
3. **시계 블록 강조** (빨간색 글로우 효과)
4. **BBC 시보음 재생**

### 4. **부드러운 페이드아웃** 🌅

알람 후 1초 뒤:
- 배경 오버레이 페이드아웃 (1초 애니메이션)
- 카운트다운 숫자 제거
- 모든 효과 초기화

## 🔧 기술적 세부사항

### Web Audio API 사용
```javascript
// 6번의 짧은 비프 + 1번의 긴 비프
for (let i = 0; i < 6; i++) {
  playBeep(audioContext, now + i * 0.8, 0.1, 1000, 0.4);
}
playBeep(audioContext, now + 6 * 0.8, 1.0, 1000, 0.5);
```

### CSS 애니메이션
```css
@keyframes pulse {
  0%, 100% { transform: translate(-50%, -50%) scale(1); }
  50% { transform: translate(-50%, -50%) scale(1.1); }
}

@keyframes flash-effect {
  0% { background-color: rgba(255, 255, 255, 1); }
  100% { background-color: rgba(255, 0, 0, 0.3); }
}
```

### 중복 방지 메커니즘
- `lastAutoAlarmMinute` 변수로 같은 분에 여러 번 울리는 것 방지
- 초가 0보다 크면 리셋하여 다음 알람 준비
