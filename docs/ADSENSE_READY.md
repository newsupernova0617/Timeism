## 6. 🎯 AdSense 신청 절차

### Step 1: AdSense 계정 생성
1. [Google AdSense](https://www.google.com/adsense) 방문
2. "시작하기" 클릭
3. Google 계정으로 로그인
4. 사이트 URL 입력: `https://synctime.keero.site`
5. 국가 및 통화 선택

### Step 2: 사이트 연결
1. AdSense 코드 복사
2. `<head>` 태그에 삽입 (이미 완료)
3. "사이트에 코드가 있습니다" 체크

### Step 3: 검토 대기
- 일반적으로 1~2주 소요
- 이메일로 승인/거절 통지

### Step 4: 승인 후 작업
1. 광고 단위 생성
2. 광고 코드를 사이트에 배치
3. 결제 정보 입력
