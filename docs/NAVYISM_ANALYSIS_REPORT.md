## 1. 서비스 개요

네이비즘(Navyism)은 특정 웹사이트의 **서버 시간을 실시간으로 확인**해주는 서비스입니다. 

### 주요 사용 목적
- **공연 티켓팅**: 콘서트, 뮤지컬 등 티켓 오픈 시 정확한 시간 동기화
- **대학 수강신청**: 수강신청 서버와의 시간 동기화
- **쇼핑 이벤트**: 한정 판매, 선착순 이벤트 참여

### 서비스 특징
- 0.001초(밀리초) 단위의 정밀한 시간 표시
- 티켓팅 연습 도구 및 커뮤니티 기능 제공
- 실시간 접속자 통계 및 트렌드 분석

---

## 2. 핵심 기능: 서버 시간 조회

### 2.1 실시간 시간 표시
- 현재 날짜와 시간(시, 분, 초)을 **대형 텍스트**로 실시간 표시
- 사용자의 로컬 시간이 아닌 서버 기준 시간 제공

### 2.2 특정 사이트 서버 시간 조회
- 상단 검색창에 URL 입력
- **'Fight!'** 버튼 클릭 시 해당 사이트 서버의 정확한 시간 조회
- 티켓팅 사이트와의 시간 동기화 지원

### 2.3 인기 사이트 목록
메인 화면 하단에 현재 접속자가 많은 인기 사이트 표시:
- 인터파크
- 네이버
- 멜론티켓
- 예스24
- 기타 인기 사이트

각 사이트별 **실시간 접속자 수** 표시

## 4. 상단 메뉴 기능

### 4.1 표준시간
- **URL**: 메인 페이지
- 기본 서버 시간 확인 페이지
- 서비스의 핵심 기능 제공

### 4.2 네이비즘 댓글
- **URL**: `/bbs.php`
- 사용자들이 티켓팅 성공을 기원하거나 정보를 공유하는 **커뮤니티 공간**
- 실시간 댓글 기능 제공

### 4.3 카운트다운 ⏱️
- **URL**: `/countdownList.php`
- 수능, 시험, 주요 행사 등 특정 시점까지 남은 시간을 카운트다운 형태로 표시
- 개인화된 카운트다운 **생성 및 공유** 가능

### 4.4 포도알 만들기 🍇
- **URL**: `/podoal`
- 티켓팅 좌석 선택 연습을 위한 **가상 좌석 배치도 도구**
- 사용자가 직접 '포도알'(좌석)을 그려보고 배치 가능

### 4.5 티켓팅 연습게임 🎮
- **URL**: `/game`
- 무작위로 나타나는 좌석(포도알)을 빠르게 클릭하는 **미니게임**
- 클릭 속도와 정확도 연습 가능
- 실제 티켓팅 상황에 대비한 반응 속도 훈련

### 4.6 생일축하시간 🎂
- **URL**: `/birthday.php`
- 특정인의 생일을 축하하기 위한 **전용 카운트다운 페이지**
- 축하 음악 연주 기능 포함
- 맞춤형 생일 축하 페이지 생성 가능

### 4.7 설문조사 📊
- **URL**: `/surveyList.php`
- 연말 소망, 승리 예상 팀 등 다양한 주제의 **설문 기능**
- 사용자 참여형 콘텐츠

### 4.8 네이비즘트렌드 📈
- **URL**: `/report.php`
- **AI 분석 기반** 당일 인기 사이트 및 주요 트렌드 보고서 제공
- 실시간 접속 데이터 기반 분석

### 4.9 네이비즘어워즈 🏆
- **URL**: `/awards.php`
- 연간 가장 인기 있었던 사이트들을 **카테고리별 순위**로 제공
- 카테고리: 티켓팅, 수강신청, 쇼핑 등
