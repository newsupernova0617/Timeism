   - Windows 11 상세 가이드

4. 브라우저의 시간 처리 방식
   - JavaScript Date 객체의 한계
   - performance.now()의 활용
   
5. SyncTime의 기술적 접근
   - HTTP HEAD 요청 분석
   - RTT 보정 알고리즘 설명
